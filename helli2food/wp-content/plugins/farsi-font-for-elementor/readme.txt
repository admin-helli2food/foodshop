== Description ==

این افزونه, فونت فارسی فونت فارسی وزیر را به صفحه‌ساز المنتور شما اضافه می‌کند. میتوانید از تنظیم تایپوگرافی المنتور, فونت وزیر رو انتخاب کنید.
اگر قالب‌ای که استفاده میکنید استاندارد باشد, می توانید فونت تمام سایت را با یک کلیک تغییر دهید. برای انجام اینکار از ویرایشگر المنتور وارد "تنظیمات سایت" و "فونت‌های عمومی" بشید.
